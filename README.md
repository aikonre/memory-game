# 🧠 Memory Card Game

A simple memory card game built with **React** and **Tailwind CSS** using **Vite**.

## 🎮 Features
- Easy / Medium / Hard game modes
- Click tracker and match counter
- Reset & back-to-menu button
- Clean responsive UI with animation

## 🚀 Tech Stack
- React
- Tailwind CSS
- Vite
